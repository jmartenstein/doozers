# User Journey Mapping

A journey map is a visual representation of the process a person goes through to accomplish a goal.

## Core Concepts

- **Visualization:** Mapping the user's process, thoughts, and emotions.
- **Shared Vision:** Aligning teams on a research-based understanding of user experience.
- **Empathy-Centric:** Focus on the user's actions and mindsets at every touchpoint.
- **Evidence-Based:** Ground maps in qualitative and quantitative research.

## Workflow for Mapping

1. **Define Scope and Actor:** Choose a specific persona and scenario.
2. **Gather Research:** Understand the user's actual path via interviews and analytics.
3. **Define Phases:** Break the journey into stages (e.g., Awareness, Purchase, Support).
4. **Map Actions, Mindsets, and Emotions:** Detail touchpoints and emotional states.
5. **Identify Opportunities:** Pinpoint pain points for innovation and improvement.

## Guidelines for Developers

- **Keep it Specific:** Focus on one persona and scenario per map.
- **Collaborative Creation:** Build with stakeholders from different departments.
- **Focus on the "Why":** Understand user motivations at each step.
- **Actionable Outcomes:** Prioritize improvements and strategic insights.
